# :mahjong: Domino Game (Classic Monte Carlo)
Классическая реализация игры в домино на C++ с графическим интерфейсом (raylib) и ботом на основе Monte Carlo.

---
## Особенности
- Полноценная игра в домино (28 костяшек, правила "до козла")
- Графический интерфейс: перетаскивание костяшек, визуализация поля
- Бот с простым случайным противником
- Сохранение статистики (победы, рекорды) в JSON
- Кроссплатформенность (Windows, Linux, macOS через raylib)

---
## Алгоритм MC
Бот использует Monte Carlo Simulation:
Для каждого возможного хода запускает N случайных симуляций; выбирает ход, который чаще всего приводит к победе.
Простая, но рабочая стратегия для начала игры

---
## Технологии
- C++20
- raylib — для 2D-рендеринга и ввода
- nlohmann/json — для сохранения статистики
- CMake — система сборки
