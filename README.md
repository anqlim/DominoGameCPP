# :mahjong: Domino Game (MCTS AI + Optimizations)
Высокопроизводительная версия игры в домино с интеллектуальным ботом на основе MCTS и набором системных оптимизаций. Проект демонстрирует применение промышленных методов ускорения C++-кода.

---
## Ключевые улучшения

- MCTS (Monte Carlo Tree Search) вместо простого Monte Carlo
  - Умный выбор ходов через баланс эксплуатации и исследования
  - Сбор статистики в дереве решений
  - Качество игры на порядок выше при меньшем времени
- Оптимизированная отрисовка
  - Предварительная генерация текстур для костяшек
  - Автоматическое масштабирование под размер экрана
  - Отказ от примитивов в пользу GPU-рендеринга
- Системные оптимизации
  - Флаги компилятора: `-O3 -march=native -DNDEBUG -fno-rtti -fno-exceptions`
  - Кэширование (хэши текстур)

---
## Алгоритм MCTS
1. **Selection:** спуск по дереву с выбором узла по формуле UCB1
$$
\text{UCB1} = \frac{w_i}{n_i} + c \sqrt{\frac{\ln N}{n_i}}
$$
где $w_i$ — победы, $n_i$ — посещения, $N$ — посещения родителя.
2. **Expansion:** добавление новых дочерних состояний
3. **Simulation:** короткая (10 ходов) случайная симуляция
4. **Backpropagation:** обновление статистики побед/посещений
